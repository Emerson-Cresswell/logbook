================================================================================
1. CURRENT STATE SUMMARY (authoritative)
================================================================================

MyLogbook is a static local-first medical procedure/CPD logbook PWA.

Current architecture:
- HTML/CSS/JavaScript (no build tool)
- service worker + manifest
- localStorage data model
- manual JSON backup/import
- Excel export via xlsx.full.min.js
- Firebase Hosting + GitHub Actions PR preview/live workflows

Current workflow:
1. Create/update a feature branch.
2. Open/update PR against main.
3. GitHub Actions run syntax + Firebase PR preview.
4. Browser check on PR preview URL.
5. Installed-PWA checks on https://test.mylogbook.uk.
6. Fix on same branch/PR.
7. Merge when checks pass and user approves.
8. Update HANDOVER/CHANGELOG/TESTING_CHECKLIST for meaningful changes.
